# Memory Game

This is a simple memory game where the player must find matching pairs.

Here's a [working demo.](https://enthusiastic-jellyfish.surge.sh)

## Running

```sh
npm install
npm start
```

## Testing

Make sure the development server is on, and run:

```sh
npm run cypress:open
````
